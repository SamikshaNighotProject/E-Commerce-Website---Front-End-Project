# E-Commerce-Website---Front-End-Project


This is a static front-end e-commerce website built using HTML5 and CSS3. 
The project demonstrates the structure and visual design of an online shopping platform, 
including product display sections, navigation menus, and responsive layout.

## Features
- Clean and user-friendly UI
- Responsive design for mobile and desktop
- Product listing and category sections
- Navigation bar (Home, Products, About, Contact)

## Technologies Used
- HTML5
- CSS3
